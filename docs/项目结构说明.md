## 📝 文件说明

### 前端文件 (src/)

#### 路由系统 (routes/)
- SvelteKit 基于文件系统的路由
- `+layout.svelte`: 全局布局组件
- `+page.svelte`: 首页
- `voice-clone/+page.svelte`: 配音生成页面
- `sample-library/+page.svelte`: 样音库管理页面
- `subtitle/+page.svelte`: 字幕生成页面
- `file-manager/+page.svelte`: 文件管理页面

#### 组件库 (lib/components/)
- `common/`: 通用组件（按钮、输入框、卡片等）
- `audio/`: 音频相关组件（播放器、波形图等）
- `form/`: 表单组件

#### 状态管理 (lib/stores/)
- `user.ts`: 用户登录状态、用户信息、Token 管理（使用 Svelte stores）
- `engine.ts`: 当前使用的引擎、可用引擎列表
- `audio.ts`: 音频生成状态、进度、当前音频路径

#### 工具函数 (lib/utils/)
- `api.ts`: Tauri 命令调用的封装函数
- `format.ts`: 文件大小、时长、日期格式化
- `constants.ts`: 应用常量定义

### 后端文件 (src-tauri/)

#### 入口文件
- `main.rs`: Rust 应用入口，注册 Tauri 命令和插件

#### 命令 (commands/)
- `audio.rs`: 音频生成、参考音频处理命令
- `engine.rs`: 引擎切换命令
- `file.rs`: 文件列表、删除命令
- `user.rs`: 用户登录命令

#### 服务 (services/)
- `tts/`: TTS 引擎实现
  - `lightweight.rs`: 轻量引擎
  - `emotion.rs`: 情感引擎
  - `cloud.rs`: 云端引擎
- `audio.rs`: 音频处理服务（裁剪、合并、格式转换）
- `subtitle.rs`: 字幕生成服务

#### 工具 (utils/)
- `config.rs`: 配置管理（用户会话、路径记忆）
- `logger.rs`: 日志系统
- `path.rs`: 路径处理工具

### 配置文件

#### 前端配置
- `package.json`: Node.js 依赖和脚本
- `jsconfig.json`: JavaScript 配置（SvelteKit 使用）
- `svelte.config.js`: SvelteKit 配置
- `vite.config.js`: Vite 构建配置

#### 后端配置
- `Cargo.toml`: Rust 依赖配置
- `tauri.conf.json`: Tauri 应用配置（窗口、插件、打包等）
- `build.rs`: Tauri 构建脚本
