## 📝 文件说明

### 桌面端前端 (frontend/src/)

#### 路由系统 (routes/)
- SvelteKit 基于文件系统的路由
- `+layout.svelte`: 全局布局（侧边栏导航 + 自定义标题栏 + 内容区）
- `+layout.ts`: SSR 禁用配置
- `+page.svelte`: 首页（Banner 轮播、公告、快捷入口、用户中心）

#### 组件 (lib/components/)
- `Sidebar.svelte`: 侧边栏导航组件
- `TitleBar.svelte`: 自定义标题栏（登录/注册弹窗、用户信息、窗口控制）
- `MenuItem.svelte`: 菜单项组件
- `Tooltip.svelte`: 提示框组件

#### 图标 (lib/icons/)
- `Icon.svelte`: 统一 SVG 图标组件（内置所有图标定义）

### Tauri 后端 (frontend/src-tauri/)

#### 入口文件
- `main.rs`: Rust 应用入口
- `lib.rs`: Tauri 应用配置（命令注册、插件加载、窗口设置）

#### 命令 (commands/)
- `user.rs`: 用户认证命令（登录、注册、Token 刷新、用户信息获取）
- `tts.rs`: TTS 生成命令
- `app.rs`: 应用级命令
- `audio.rs` / `engine.rs` / `file.rs` / `sample.rs`: 待实现

#### 服务 (services/)
- `tts/mod.rs`: TTS 引擎服务模块（待实现）

#### 工具 (utils/)
- `api.rs`: 后端 API HTTP 请求封装（auth.wzagent.cn）
- `config.rs`: 配置管理
- `storage.rs`: 本地存储（Token 持久化到文件系统）

### 管理后台 (admin_panel/)

#### API 后端 (api/)
- **技术栈**: PHP 8.2，原生路由，PDO 数据库
- **入口**: `index.php`（路由分发 + 频率限制 + 安全检查）
- **控制器**: AdminApiController（管理操作）、AuthController（用户认证）、UserController（用户端）、BannerController（Banner）
- **公共库**: Database（PDO 封装）、JWTAuth（JWT 认证）、SecurityGuard（安全防护）

#### 管理前端 (admin-frontend/)
- **技术栈**: Vue 3 + Element Plus + Vite
- **页面**: 控制台、用户管理、卡密管理、字符包管理、营销活动、分佣提现、Banner 管理、公告管理、软件配置、系统设置、操作日志

#### WebSocket 服务 (websocket/)
- **技术栈**: PHP Ratchet
- **功能**: 用户实时状态同步、强制登出推送

### 参考引擎 (IndexTTS/)

> ⚠️ **核心原则**: IndexTTS 仅作为丸子配音引擎开发的参考资料，**禁止直接调用**

- `v15/`: IndexTTS 1.5 轻量引擎源码（tag v1.5.0）
- `v20/`: IndexTTS 2.0 情感引擎源码（main 分支）
- 已在 `.gitignore` 中排除，不提交到 Git

### 配置文件

#### 前端配置 (frontend/)
- `package.json`: Node.js 依赖和脚本
- `tsconfig.json`: TypeScript 配置
- `svelte.config.js`: SvelteKit 配置（SSR 适配 + Tauri 集成）
- `vite.config.js`: Vite 构建配置

#### 后端配置 (frontend/src-tauri/)
- `Cargo.toml`: Rust 依赖配置
- `tauri.conf.json`: Tauri 应用配置（窗口、标识符、插件）
- `build.rs`: Tauri 构建脚本
