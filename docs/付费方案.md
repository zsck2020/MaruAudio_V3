# 丸子配音 V3 · 付费方案

> **最后更新**: 2026-05-27  
> **策略结论**: 免费试用建立信任，永久旗舰吃下个人创作者，云端字符包承接无显卡与批量需求，团队授权服务机构客户。  
> **对外表达原则**: 只说「轻量引擎 / 情感引擎 / 云端引擎」，不在营销页暴露底层模型、部署方和内部技术名。

---

## 一、市场判断

### 1.1 用户需求变化

2026 年 AI 配音已从「把文字念出来」变成「内容生产流水线的一环」。丸子配音应围绕创作者的真实工作流收费，而不是只按模型能力收费。

| 用户群 | 核心需求 | 付费触发点 | 丸子承接方式 |
|---|---|---|---|
| 短视频创作者 | 快速出片、情绪自然、低成本试错 | 每天都要生成，厌烦多平台订阅 | 免费日额度 + 旗舰本地无限 |
| 短剧 / 漫剧团队 | 多角色、台词级情绪、批量重试 | 长文本、多角色、交付周期紧 | 多角色项目 + 行级编辑 + 批量生成 |
| 有声书 / 播客 | 长内容稳定输出、音色一致 | 长文本拆分、术语读法、字幕对齐 | 智能拆分 + 术语词汇表 + 字幕工作流 |
| 个人 IP / 知识博主 | 专属声线、商用授权、私密素材 | 声音克隆与商业使用 | 本地克隆 + 无水印导出 |
| 无显卡用户 | 不想折腾环境，笔记本也要高质量 | 本地跑不动或太慢 | 云端字符包 |
| 工作室 / 代运营 | 批量交付、多人设备、发票与售后 | 团队协作与稳定服务 | 团队授权 + 大额字符包 |

### 1.2 主流计费趋势

| 模式 | 市场代表 | 用户感受 | 风险 | 丸子取舍 |
|---|---|---|---|---|
| 月订阅 + 月额度 | ElevenLabs / Murf / LOVO | 入门便宜，但不用也扣费 | 订阅疲劳、额度过期 | 不作为主付费锚点 |
| 积分 / 字符包 | 云厂商 API / 创作平台 | 用多少买多少，可控 | 价格不透明会劝退 | 用于云端引擎，必须永不过期 |
| 一次买断 | 桌面工具 / 本地工具 | 决策简单，口碑传播强 | 后续收入不足 | 作为丸子核心差异化 |
| 免费开源自部署 | GPT-SoVITS / SonicVale 类 | 零软件费 | 环境门槛高、维护成本高 | 用「开箱即用」打差异 |
| 企业定制 | 云厂商 / API 平台 | 服务稳定，可开发票 | 销售周期长 | 后置，不影响 C 端简单定价 |

### 1.3 丸子的商业定位

**一句话定位：给中文创作者的本地优先 AI 配音工作台。**

丸子不应与纯云端 SaaS 正面对拼「海量音色库」，也不应与开源项目比「免费」。最强卖点是：

- **本地优先**：素材、样音、项目默认留在本机，创作者更放心。
- **一次买断**：本地引擎无限用，避开订阅疲劳。
- **情感与多角色**：面向短剧、有声书、剧情类短视频，而不是普通朗读。
- **云端兜底**：没有独显或临时批量任务时，用字符包补齐体验。
- **桌面开箱即用**：比开源自部署省时间，比网页工具更像生产力软件。

## 二、套餐总览

### 2.1 推荐定价结构

| 套餐 | 价格 | 目标用户 | 核心价值 |
|---|---:|---|---|
| 免费版 | ¥0 | 首次体验、轻量用户 | 完整体验基本流程，制造升级欲望 |
| 旗舰版 | ¥198 永久 | 个人创作者、博主、播客 | 本地双引擎无限生成，无水印商用 |
| 创作者年卡 | ¥199 / 年 | 不想买字符包、希望持续云端额度的用户 | 旗舰权益 + 每月云端赠额 + 优先更新 |
| 团队版 | ¥999 / 年 / 3 设备 | 工作室、代运营、小团队 | 多设备授权、商用凭证、批量字符折扣 |
| 云端字符包 | ¥9.9 起 | 无显卡、临时批量、长内容 | 按量购买，永不过期，只扣云端 |

> 建议上线顺序：先做「免费版 + 旗舰版 + 云端字符包」，年卡与团队版等支付闭环稳定后再开放，避免早期产品面过复杂。

### 2.2 为什么保留 ¥198 永久

| 对比项 | 月订阅 | ¥198 永久 |
|---|---|---|
| 首次决策 | 看似低，但担心长期扣费 | 一次决策，容易理解 |
| 用户口碑 | 「又一个订阅」 | 「买一次就能本地无限用」 |
| 适合场景 | 云端持续消耗 | 桌面工具、本地生成 |
| 收入持续性 | 强 | 需靠字符包和团队版补充 |
| 丸子匹配度 | 中 | 高 |

¥198 应作为公开主锚点。它的心理账本是：

- 低于多数创作者一年订阅成本。
- 与「本地无限生成」强绑定。
- 能降低用户对云端字符扣费的戒心。
- 适合短视频平台用一句话传播：「198 买断，不限本地生成」。

## 四、云端字符包

### 4.1 字符包定价

> **规则**：只扣云端引擎，本地轻量 / 情感引擎永不扣字符。

| 字符包 | 价格 | 单价 | 适合场景 |
|---|---:|---:|---|
| 体验包 · 1 万字 | ¥9.9 | ¥0.99 / 千字 | 首次体验云端品质 |
| 日常包 · 5 万字 | ¥39.9 | ¥0.80 / 千字 | 短视频日常配音 |
| 创作包 · 20 万字 | ¥129 | ¥0.65 / 千字 | 有声书、短剧、长内容 |
| 商业包 · 100 万字 | ¥499 | ¥0.50 / 千字 | 工作室批量交付 |
| 团队包 · 500 万字 | ¥1,999 | ¥0.40 / 千字 | 机构客户、内容工厂 |

### 4.2 字符包规则

- **永不过期**：单独购买的字符永久有效，降低购买阻力。
- **可叠加**：余额累加，优先消耗即将过期的赠送额度。
- **只扣云端**：本地生成不扣字符。
- **生成前预估**：点击生成前展示预计扣减字符。
- **失败返还**：云端生成失败、取消、超时未产出时自动返还。
- **重试优惠**：同一段文本短时间内重试，可设置 1 次免费或半价扣减。
- **退款规则**：72 小时内且未使用的字符包可原路退款。

### 4.3 云端引擎文案

对用户只表达：

- 「电脑配置不够也能生成高质量配音」
- 「批量任务不占本地显卡」
- 「和情感引擎同级效果，适合长文本与赶稿」

避免表达：

- 底层模型名称
- 服务器部署方
- 推理镜像、框架、API 细节

## 六、转化策略

### 6.1 免费到旗舰的五个转化触点

| 触点 | 触发场景 | 推荐话术 | 产品动作 |
|---|---|---|---|
| 情感引擎锁定 | 用户切换到情感引擎 | 「同一句台词，情感引擎更像真人表演」 | 播放 A/B 试听 |
| 字符耗尽 | 当日 3,000 字用完 | 「今天额度已用完，升级后本地生成不限字数」 | 展示倒计时 + 升级按钮 |
| 水印感知 | 导出后听到尾部水印 | 「旗舰版导出无水印，可直接商用」 | Toast + 一键升级 |
| 多角色需求 | 添加第二个角色 | 「短剧 / 对话内容需要多角色配音」 | 展示多角色示例项目 |
| 长文本导入 | 导入长稿或批量生成 | 「长内容建议使用批量生成和失败重试」 | 弹出效率对比 |

### 6.2 旗舰到云端字符包的三条路径

| 用户画像 | 判断条件 | 推荐入口 |
|---|---|---|
| 无独显用户 | 本地生成速度慢、设备检测无独显 | 「用云端引擎立即生成」 |
| 批量赶稿用户 | 单次文本超过 2 万字或连续批量失败 | 「本地继续跑，同时用云端加速」 |
| 多设备用户 | 经常换电脑、笔记本办公 | 「云端不占本机资源」 |

### 6.3 升级弹窗原则

1. 第一次点击锁定功能才弹窗，之后降级为轻提示。
2. 弹窗必须展示「用户刚刚想做的事」，不要泛泛列功能。
3. 情感引擎相关弹窗必须有试听，不要只放文字。
4. 价格锚点固定为「¥198 永久，本地不限」。
5. 不用恐吓式倒计时，限时优惠只在首购和活动期使用。
6. 支付不要跳出太远，尽量在应用内完成扫码。

### 6.4 首次启动引导

```text
首次启动
  ↓
选择目标：短视频 / 有声书 / 短剧 / 播客 / 个人声音
  ↓
预置一段示例文本和推荐参数
  ↓
生成免费版音频
  ↓
播放情感引擎示例试听
  ↓
提示：升级后本地无限生成、无水印、可商用
  ↓
进入正常工作台
```

## 十、风险与应对

| 风险 | 等级 | 应对 |
|---|---|---|
| 用户觉得 ¥198 贵 | 中 | 首发 ¥168 + 试听对比 + 年成本对比 |
| 免费版太弱导致流失 | 中 | 保证能完整生成、试听、导出，只限制高价值能力 |
| 免费版太强影响付费 | 中 | 锁定多角色、批量、情感精调、无水印商用 |
| 云端字符包使用率低 | 中 | 赠送少量体验字符，生成慢时智能推荐云端 |
| 云端成本失控 | 高 | 先冻结字符，失败返还，限制异常重试 |
| 盗版 / 共享账号 | 高 | 机器码绑定、设备数限制、关键权益服务端校验 |
| 断网误伤付费用户 | 中 | 会员状态本地缓存，允许短期离线使用 |
| 营销承诺过满 | 高 | 不承诺「真人不可分辨」，只承诺效率、隐私、可控 |
| 底层技术暴露 | 中 | 对外统一使用三类引擎命名 |
