# 逗哥配音功能对标 — 丸子配音 (IndexTTS)

> 逗哥配音功能中，IndexTTS v1.5 / v2.0 可实现的部分  
> 最后更新: 2026-03-18

## 可实现功能

| 功能 | v1.5 | v2.0 | 备注 |
|------|:---:|:---:|------|
| 文本转语音 | ✅ | ✅ | 核心能力 |
| 声音克隆 | ✅ | ✅ | 零样本克隆，5秒参考音频 |
| 多人对话 | ✅ | ✅ | 切换参考音频，有缓存机制 |
| 拼音标注 / 多音字 | ✅ | ✅ | 字符-拼音混合建模，IndexTTS 特色 |
| 声调标记 | ✅ | ✅ | MA1/MA2/MA3/MA4 数字标注 |
| 数字读法 | ✅ | ✅ | TextNormalizer 内置中文数字转换 |
| 儿化音 | ✅ | ✅ | 训练数据天然支持 |
| 插入停顿 | ✅ | ✅ | 标点符号控制：逗号=短停，句号=中停 |
| 感叹词（嗯、啊） | ✅ | ✅ | 文本中插入语气词，引擎正常合成 |
| 文义分段 | ✅ | ✅ | TextTokenizer 按标点和 token 自动分段 |
| 导入文案 | ✅ | ✅ | 纯前端功能 |
| 情感调节（喜怒哀乐） | — | ✅ | v2.0 支持 8 维情感向量 |
| 宣传配音风格 | — | ✅ | v2.0 情感向量 + 参考音频配合 |
| 语速调节 | ✅ | ✅ | FFmpeg atempo 后处理变速不变调 |

## 丸子配音差异化优势

| 丸子配音 | 逗哥配音 |
|---------|---------|
| 本地离线运行 | 依赖云端 |
| 数据不出设备 | 文本/音频上传服务器 |
| 一次部署无限用 | 按字数付费 |
| 拼音混合建模精确控制 | 标准 TTS |
| v2.0 八维情感精确调节 | 预设情感风格 |
